git 管理项目 总结

这两天主要学习了git的相关使用方法和步骤，通过git将我们自己的文件提交到云端，以避免电脑出现错误时丢失文件。下面是git的一个操作方法:

1.安装软件，如环境变量不符，需要配置环境变量；

2.安装完成后，就可以进行项目的提交等操作；

3.新建一个文件夹，在空白处按住shift,然后右击 在此处打开命令窗口 就可以进行操作了

4.首先需要用 git init 来初始化仓库；

5.然后用 git add yourfilesname.yourfilestype(例如：test.txt/index.html...and so on...)来提交你的单个文件；

6.git add . 将所有文件和文件夹提交到待提交文件的队列中；

7.用git commit 将文件提交到版本库中；

8.在上一步最后加上 -m 添加提交到版本库的版本描述信息；

9.最后一步：git push origin master 将文件提交到云端版本库(会出现输入用户名和密码)，然后完成文件的提交。

上述是用git将文件提交到云端的步骤。

备份本地项目的操作方法如下：

1.注册登录云端git服务器；

2.在云端创建一个备份项目；

3.克隆云端项目到本地:git clone httpsaddress(你的地址)

4.复制本地项目到git 项目中；

5.用cd 添加你所要加的项目目录；

6.git add . 将所有文件和文件夹提交到待提交文件的队列中；

8.git commit -m "描述":提交到版本库；

9.最后一步：git push origin master 将文件提交到云端版本库(会出现输入用户名和密码)，然后完成文件的提交。

上述是用git将文件提交到云端的步骤。
